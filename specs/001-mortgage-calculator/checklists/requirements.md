## Notes

- **2025-12-05**: 初版規格建立，包含 3 個 User Stories（基本試算、金額/成數切換、預設值）
- **2025-12-05**: 新增利率類型切換功能（固定/浮動利率），更新為 4 個 User Stories
- **2025-12-05**: 新增即時計算結果與可視化功能，更新為 5 個 User Stories（P1）
  - 新增功能：總還款金額、總利息金額、首月本金/利息比例、還款曲線折線圖、即時更新
  - 新增 FR-016 ~ FR-021（輸出相關）
  - 新增 SC-006 ~ SC-008（效能與可用性）
  - 新增 Edge Cases：圖表資料聚合、視窗自適應、極端值顯示
  - 新增 Key Entities：首月還款明細、還款曲線資料
- **2025-12-05**: 新增還款明細表與數字格式化功能，更新為 7 個 User Stories
  - 新增 User Story 6：完整還款明細表（P1）- 每期顯示期數、應付總額、本金、利息、剩餘本金
  - 新增 User Story 7：數字格式化與可讀性（P2）- 千分位格式、比較說明文字
  - 新增 FR-025 ~ FR-029（還款明細表相關）
  - 新增 FR-030 ~ FR-033（數字格式化相關）
  - 新增 SC-009 ~ SC-011（明細表效能與格式化）
  - 新增 Edge Cases：長期數捲動效能、四捨五入誤差處理、大數字格式化
  - 新增 Key Entities：還款明細表、格式化設定
  - 新增 Assumptions：千分位格式使用逗號、金額取整數位
